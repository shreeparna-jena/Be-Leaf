# Be-Leaf

A gamified carbon footprint awareness platform that helps users understand, track, and reduce their environmental impact through personalized insights, Seed Points, and growth-based sustainability goals.

## Features

- Friendly onboarding with O'Leaffy 🍀
- Carbon footprint awareness guide
- Personalized sustainability questionnaire
- Footprint Score and Seed Score system
- Growth progression:
  Seed → Sapling → Plant → Tree → Garden
- Personalized recommendations
- Sustainability leaderboard
- Responsive interface

## Tech Stack

- React
- Vite
- JavaScript
- CSS

## Purpose

Built for the Carbon Footprint Awareness Platform challenge.
